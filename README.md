This is an accompanying github repository for the review paper "Alternatives to the Cox proportional hazards model: A review of additive hazards and accelerated failure time models", by Hein Putter and Liesbeth de Wreede, for publication in Statistica Neerlandica. It contains the material for Section 5 of the paper. Hazard_comparison (R file and pdf) contain R code and results illustrating behaviour of proportional hazards, additive hazards and AFT models, in terms of underlying hazards, cumulative hazards and survival functions. Illustration_pbc (R file and pdf) contain R code and results of fitting these three different models on the PBC data, using two binary covariates.
